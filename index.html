Shurity

Shurity was designed for a startup client as a demo. The focus of the startup is to help people purchase electronic equipment without a confusing babble of specs. The app presents a series of simple questions (various slider bars and dropdowns on one page) and then produces a set of recommendations. The user can revise the answers and get a different set of recommendations. This project was built with Ruby on Rails and PostgreSQL with limited Javascript and JQuery.
